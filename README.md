## Contributing

- Fork the repo
- Clone the repo
- Install the dependency:
  ```bash
  npm install
  npm install -g ngrok
  ```
- Create copy `.env.example` to `.env` and replace the environment variable
- Start the server
  ```bash
  npm run dev
  ```
